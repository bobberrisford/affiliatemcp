# Ready-to-publish LinkedIn posts

> Companion to [`social-video-playbook.md`](./social-video-playbook.md). Four
> complete posts: the visual asset, the caption, the first comment (where the
> link goes, to protect reach), and hashtags. Voice is plain and concrete, UK
> English, hook on the first line. Swap the GitHub link for the live site if you
> have one. Data in the visuals is scrubbed (Acme / CashbackCo, demo numbers).

## Posting notes

- Put the link in the first comment, not the post body, to avoid the
  outbound-link reach penalty.
- The first caption line is the hook; it must work before LinkedIn's "…more"
  truncation.
- One post a week, rotating through the pillars in the playbook, keeps the
  format consistent.
- Re-run any asset with `node scripts/social-video.mjs <post-id> [video|carousel|static]`.
