# Everflow (Publisher / Affiliate side)

**Claim status:** experimental  
**Adapter version:** 0.1.0  
**Last verified:** 2026-05-28  
**API docs:** <https://developers.everflow.io/docs/affiliate/>

---

## Overview

Everflow is a performance marketing platform used by affiliate networks and brands to manage and track affiliate campaigns. This adapter integrates with the **Affiliate API** (publisher side), allowing you to list visible offers, retrieve conversion reports, generate tracking links, and query click data.

> **Important:** Everflow affiliate API keys are generated by the **network admin**, not self-service by the affiliate. You must contact your Everflow network operator and request an API key before setup can complete.

---

## Prerequisites

- An active affiliate account on an Everflow-powered network.
- Your network admin must generate an API key for your account under **Manage Affiliate → API tab**.
- The network admin will hand you the key (it is only shown once at generation time).

---

## Setup

### 1. Obtain your API key from the network admin

1. Contact your Everflow network operator.
2. Ask them to navigate to **Manage Affiliate → (your account) → API tab**.
3. They click **"+ API Key"** to generate a new key.
4. They copy the key immediately (the full value is only shown once) and send it to you securely.

> Keys are long-lived and scoped to your affiliate account by the admin. If a key is compromised, ask the admin to revoke it and issue a new one.

### 2. Configure credentials

Run the setup wizard:

```bash
affiliate-networks-mcp setup everflow
```

Or add credentials manually to `~/.affiliate-mcp/.env`:

```env
EVERFLOW_API_KEY=ef_aff_xxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxx
EVERFLOW_AFFILIATE_ID=12345   # optional — for richer diagnostic output
```

### 3. Verify the connection

```bash
affiliate-networks-mcp doctor everflow
```

Expected output: `✓ everflow: authenticated as everflow/affiliate/12345`

## Supported operations

| Operation | Status | Notes |
|---|---|---|
| `listProgrammes` | Experimental | Lists all visible offers via `GET /v1/affiliates/alloffers`. |
| `getProgramme` | Experimental | Fetches a single offer by numeric ID via `GET /v1/affiliates/offers/{offerId}`. |
| `listTransactions` | Experimental | Raw conversion report via `POST /v1/affiliates/reporting/conversions`. |
| `getEarningsSummary` | Experimental | Derived from `listTransactions` — aggregates commissions by programme and status. |
| `listClicks` | Experimental | Click stream via `POST /v1/affiliates/reporting/clicks/stream`. Maximum 14 days per call; wider windows are chunked automatically. |
| `generateTrackingLink` | Experimental | Calls `GET /v1/affiliates/offers/{offerId}/url/0` to fetch a tracking URL from the server. |
| `verifyAuth` | Experimental | Cheap probe via `GET /v1/affiliates/alloffers?page=1&page_size=1`. |

## Known limitations

- **Adapter built from public API documentation; not yet verified against a live account.** All endpoint shapes carry `// TODO(verify)` annotations on uncertain fields.
- **Affiliate API keys must be created by a network admin**, not self-service by the affiliate.
- **Click stream caps at 14 days per call.** The adapter automatically chunks wider windows into ≤14-day slices, so this is transparent to callers.
- The `currency` field on offers is a numeric `currency_id` in Everflow's schema; ISO currency mapping is not available without a lookup table. Currency defaults to `USD` on conversions.

## API reference

| Endpoint | Method | Description |
|---|---|---|
| `/v1/affiliates/alloffers` | GET | List all visible offers (paginated). |
| `/v1/affiliates/offers/{offerId}` | GET | Single offer detail. |
| `/v1/affiliates/reporting/conversions` | POST | Raw conversion report with date range filter. |
| `/v1/affiliates/reporting/clicks/stream` | POST | Raw click stream (max 14 days). |
| `/v1/affiliates/offers/{offerId}/url/{urlId}` | GET | Generate a tracking URL. urlId=0 for the default URL. |

## Common failures

1. **`config_error: Missing required credential EVERFLOW_API_KEY`**  
   Your API key has not been configured. Run `affiliate-networks-mcp setup everflow` or add `EVERFLOW_API_KEY` to `~/.affiliate-mcp/.env`.

2. **`auth_error: HTTP 401`**  
   The API key is invalid, revoked, or incorrectly copied. Contact your Everflow network admin to issue a new key.

3. **`generateTrackingLink` returns an empty tracking URL**  
   The offer may not be approved/runnable for your affiliate account. Confirm with your network admin that the offer is active for you.
