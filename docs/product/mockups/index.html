install · three ways in

five minutes. no account.

Pick the path that matches how you work. All three run the same local server with your own keys — nothing is hosted, nothing phones home.

easiest · recommended

claude desktop extension

Why: the fastest way in for Claude Desktop. Claude owns the runtime and stores your keys in secure credential fields — no Node.js, no terminal, no config files to edit.

How: download the .mcpb, then in Claude Desktop open Settings → Extensions → Advanced → Install Extension… and pick the file. Native setup fields cover Awin, CJ, Impact & Partnerize; every other network reads your existing config.

macOS + Windows · no Node.js · one-click install

get the Claude extension ▸
mac · guided wizard

macOS setup app

Why: a visual wizard for setting up several networks in one sitting. It walks each network login, stores your keys locally, wires up Claude Desktop, then quits — you ask your questions in Claude, not the app.

How: download the signed .dmg, drag to Applications, and follow the steps. Best on Apple Silicon Macs; on Windows or Intel, use the extension or the command line.

apple silicon · signed & notarised · no terminal

command line

Why: the most flexible path — works with Claude Desktop, Claude Code and Codex, on any OS, with every network. This is what powers Claude Code and Codex.

How: run the npx wizard below to add your keys, then the install command for your client. Needs Node.js 20+.
