It's open source, MIT, and built to be easy to review. Pick the one that fits — each links to live work on GitHub. You don't need Claude Code to contribute; everything here can be done by hand.

01

adopt your network's adapter

If you work for a network we ship today, take ownership of its adapter — you know your API better than the community can guess. It's the canonical path, and it can cover publisher, advertiser, or both.

ai-assisted prs are welcome

This repo ships a contribute skill so Claude Code can do the legwork — scaffolding an adapter, wiring tests, drafting setup docs. We don't mind how a PR was written; we care that it meets the bar:

SEVEN CANONICAL OPS NO FAKE SUPPORT SCRUBBED FIXTURES CI GREEN

Review SLA: first reply on external PRs within 24 hours on weekdays. If none of these fit, open an issue describing what you want to do.
