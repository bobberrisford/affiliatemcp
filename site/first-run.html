first run

you're set up.
now the first five minutes.

The networks are wired into your AI. You don't open a dashboard or learn a menu — you ask, the same way you'd ask a colleague. Start with these three. Each one calls your real networks and answers in plain English.

open Claude, start a fresh chat, and type the first one
ask these three first
is my affiliate setup working?
you getOne line per network — connected, or a plain reason why not. Run this first: it proves the wiring and tells you exactly what to fix if anything's off.
what did I earn across all my networks last month?
you getOne total, pulled live from every network you connected and split by status — pending, approved, paid, reversed — with anything unpaid past 90 days flagged. No exports, no spreadsheet.
audit the affiliate links on my site.
you getEvery link classified by network, with dead or declined programmes flagged — so revenue isn't quietly leaking through a broken link. Point it at a page, a draft, or a sitemap.
One thing to know: the AI decides which networks to call and how to answer. It works from your live data and won't invent numbers — if a network can't answer, it says so rather than guessing.
nothing happened?

the usual fixes

Almost every first-run snag is one of these. None of them need the terminal.

1Restart the app. Skills and networks load when the AI client starts. If you set up while it was open, fully quit Claude (or Codex) and reopen it, then ask again in a new chat.
2Start a fresh chat. A conversation that began before setup won't see the new tools. Open a new one and ask "is my affiliate setup working?"
3A network shows an error. That's the health check doing its job. Ask "help me set up [network]" and the AI walks you through the credential steps, quoting the dashboard menu paths.
4Still stuck? The get started page has the full setup walkthrough, and the FAQ covers the rest.
once that lands

then it starts saving you mondays

The first three are the warm-up. The point is the work the AI runs end to end — pull the numbers, draft the partner email, flag what moved, put the follow-ups in your outbox. One prompt.

Try a real one: "chase my unpaid commissions" pulls every approved-but-unpaid sale past your payment terms and drafts the chase email with them attached. Or, if you run brands, "any anomalies in the affiliate data this week?" — and Claude can run that one on a schedule, so you hear about a revenue drop before the client does.
