how it works

you don't install these one by one. you set up affiliate-mcp once, then just talk to your assistant the way you'd brief a colleague. it picks the right job for you automatically.

1

set it up once

install affiliate-mcp in Claude (about 5 minutes, we walk you through it) and connect the affiliate networks you already use.

2

ask in plain english

type what you want in your own words. you don't need to know any of the names below.

"how did Acme do last month?"
3

get the answer

your assistant pulls the real numbers, lays them out, and drafts what you need. you read it, you decide, you send.

the list below is just so you can see what's possible and learn the kinds of things to ask. browse it, or skip it and start talking. open in claude drops a prompt straight into the Claude desktop app for you to send; or copy it into Claude Code or Codex.
